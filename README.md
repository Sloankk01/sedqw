# sedqw晓彤表妹照片
第63章 他是在搞一种什么样的新型副业啊！？
等林夏睡了一觉醒来。
已经是下午两三点钟了。
起床简单洗漱了一番。
随便吃了点东西对付一下肚子。
林夏就又坐在床上陷入了沉思。
昨天夜里发生的事儿仿佛还历历在目。
讲道理，被杨蜜和热芭中间那段声音一折腾。
他感觉还是有点累的。
身体倒还好。
主要就是心理。
他都犹豫要不要去找个心理医生看看了。
因为他现在闭上眼，满脑子都是一个披头散发的老娘们。
不停地冲他招手喊个不停说：“林夏~快来啊~我们来找你了~”
就特么很淦！
林夏觉得自己已经有心理阴影了。
但转念想想现在心理医生高额的收费。
他还是打消了这个想法。
……
“虽然比较累，还造成了一点心理创伤，但挣了五千块钱，其实也不少。”
“要不，今天休息一天得了。”
林夏想了想，便不打算出门继续搞副业了。
而且，现在都已经下午两三点了。
想找活儿其实也没那么容易。
除非继续找夜班的活。
但有了昨夜的经历。
林夏觉得。
自己短时间内。
还是不要搞夜班相关的副业了。
太瘆人，万一哪天把自己吓出事儿，可就不划算了。
就算搞，也要搞那种阳间的，人气多的夜场。
倒是可以考虑考虑。
……
心中有了主意之后。
林夏就准备打开直播跟水友们请个假。
而随着他开播的一瞬间。
无数水友纷纷涌入直播间。
接着弹幕开始密密麻麻滚动起来。
“咦？竟然开播了？我以为今天主播要休息一天呢。”
“好家伙！看主播精神状态不错嘛！”
“哈哈哈，这要搁我早折腾死了，主播竟然跟没事人一样。”
“净说废话，你们也不想想，主播他是一般人吗？”
“就是！他可是喝过交杯茶的人！”
“噗哈哈哈！秀儿，你们又开始了嗷！”
“萌新报道，朋友推荐来的不懂，交杯茶是什么梗啊？”
“没什么，就是主播已经解锁了多人喝茶模式。”
“楼上的，你小子最好说的是正经多人喝茶！”
“啊这？借楼打个广告，新茶到店，欢迎品尝。”
“楼上的老哥，细嗦，全国飞多少钱？”
“……”
看着直播间如潮水般的弹幕滚动。
林夏打开后台，看了一眼关注人数。
果然，比起之前又涨了不少。
已经有150万左右的关注了。
怪不得活跃度这么高呢。
要是等以后他真正进入娱乐圈、
也有这么多粉丝就好了。
嗯……
最好全都是女粉。
甩了甩头赶走脑海中不靠谱的想法。
